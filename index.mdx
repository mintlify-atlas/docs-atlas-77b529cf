Wanman is an open-source agent matrix framework that runs a supervised network of AI agents on your local machine. Inspired by the Japanese *ワンマン電車* (one-man train), wanman lets you step back into an observer role while Claude Code or Codex agents autonomously collaborate on your codebase.

## How it works

The `wanman` CLI is a thin JSON-RPC client. All orchestration lives in a local supervisor process that manages agent subprocesses, routes messages, and persists state in SQLite.
